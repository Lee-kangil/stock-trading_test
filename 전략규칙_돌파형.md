# 전략 규칙서 — 돌파형 (연습용 #1·#5)

> 2주차 산출물. "모호함 없이 숫자로" 정의하는 것이 목표.
> 아래 숫자들은 **시작 기본값(제 추론)** 이며, 4주차 검증 단계에서 데이터로 조정합니다.
> 핵심: 사람의 판단이 끼어들 여지가 없어야 코드로 만들 수 있고, 코드로 만들어야 검증할 수 있다.

---

## 한 줄 요약
"상승 추세인 종목이, 최근 20일 고점을 새로 돌파하면 산다. 추세가 꺾이거나 손절선에 닿으면 판다."

---

## 1. 매매 유니버스 (어떤 종목을)
- 대상: 코스피200 구성 종목 (1주차엔 단일 종목으로 연습, 3주차에 확장)
- 1주차 연습 종목: 005930(삼성전자)

## 2. 진입 규칙 (언제 사는가) — 모두 만족 시 매수
1. **돌파 조건**: 오늘 종가 > 직전 20일 최고가 (`Close > High20.shift(1)`)
2. **추세 필터**: 오늘 종가 > 60일 이동평균 (`Close > SMA60`)
   - → 상승 추세에 있는 종목만 매수 (떨어지는 칼 회피)
3. 진입 체결 가정: 신호 발생일 **종가**에 매수 (단순화. 실거래는 다음 날 시가 등으로 보정 — 3주차)

## 3. 청산 규칙 (언제 파는가) — 둘 중 먼저 닿는 것
- **추세 청산(트레일링)**: 오늘 종가 < 직전 10일 최저가 (`Close < Low10.shift(1)`)
- **손절(하드 스톱)**: 종가 ≤ 매수가 × (1 − 0.07)  → 즉 **매수가 대비 −7%**

## 4. 포지션 비중 (얼마나)
- 1회 매수 = 총자산의 **10%** (최대 동시 10종목 분산)
- 1주차 단일 종목 연습에서는 비중 계산을 생략(전액 1종목)하고, 3주차 포트폴리오에서 적용

## 5. 파라미터 한눈에 (나중에 튜닝할 값)
| 항목 | 기호 | 기본값 |
|------|------|--------|
| 돌파 기준 기간 | entry_high | 20일 |
| 청산 기준 기간 | exit_low | 10일 |
| 추세 필터 | trend_ma | 60일 |
| 손절 폭 | stop_pct | 7% |
| 1회 비중 | weight | 10% |

## 6. 의사코드 (사람 말 → 코드 직전 단계)
```
매일 종가 마감 후:
  지표 계산: High20, Low10, SMA60
  만약 (보유 안 함) 그리고 (Close > 어제의 High20) 그리고 (Close > SMA60):
      매수, 매수가 기록
  아니면 만약 (보유 중):
      만약 (Close <= 매수가 * 0.93):      # 손절
          매도 (사유: 손절)
      아니면 만약 (Close < 어제의 Low10):  # 추세 청산
          매도 (사유: 추세이탈)
```

## 7. 아직 반영 안 한 것 (다음 주차)
- 수수료·거래세(0.18%)·슬리피지 → **3주차**
- 다종목 포트폴리오, 비중 배분 → 3주차
- 과최적화·미래참조 점검, Walk-forward → 4주차
